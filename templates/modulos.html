Como funcionam os módulos

Módulos de Posição

Cada posição (goleiro, lateral, zagueiro, meia, atacante, treinador) tem seu próprio módulo com pesos específicos configurados na página de configuração.

Pesos Personalizados

Cada módulo utiliza os pesos configurados individualmente, permitindo ajustes específicos para cada posição.

Análise Detalhada

Visualize todos os dados utilizados no cálculo, incluindo médias, pesos de jogo, pesos SG e estatísticas específicas.

Atualização Automática

Os dados são atualizados automaticamente a cada execução do sistema, garantindo informações sempre atuais.
