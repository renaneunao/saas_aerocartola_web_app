Associe suas credenciais do Cartola FC para começar a usar o sistema

Por que preciso adicionar um time?

Para calcular a melhor escalação e enviar automaticamente para o Cartola FC, precisamos dos tokens de autenticação da sua conta.

Suas credenciais são armazenadas com segurança e usadas apenas para acessar a API do Cartola FC.

Tokens de Autenticação

Credenciais de acesso ao Cartola FC

Token principal de acesso às APIs do Cartola FC

Token para renovação automática do access token

Token de identificação adicional (não obrigatório)

Como obter os tokens

No Navegador

  1. Acesse o Cartola FC pelo navegador
  2. Faça login na sua conta
  3. Pressione F12 (Ferramentas de Desenvolvedor)
  4. Vá na aba "Network" ou "Rede"
  5. Navegue pelo site e procure requisições com Authorization
  6. Copie os tokens dos cabeçalhos das requisições

Segurança

  • • Tokens são armazenados com segurança
  • • Não compartilhe suas credenciais
  • • Você pode atualizar a qualquer momento
